Henry's Ticker

Henry's Ticker is a lightweight, Windows-native ticker bar that attaches seamlessly to the top of the desktop as a true Windows AppBar. Built with Electron and a custom Win32 helper, it provides a persistent, always-visible strip for quick information display without interfering with normal workflows.

Features

  • True Windows AppBar: Maximized windows automatically resize between Henry and the taskbar.
  • Electron Interface: Clean UI powered by HTML, CSS, and JavaScript.
  • Installer + Portable Build: Bundled using electron-builder.
  • Native Icon Support: Fully branded taskbar/start-menu icon.
  • Settings Window: Quick adjustments for bar size and position.
  • Lightweight C++ Helper: Handles AppBar registration via Win32 APIs.

Technology Stack

  • Electron 39
  • Node.js 18
  • C++ Win32 API
  • electron-builder
  • MinGW64 for compiling native helper

Build Instructions

  1. Install dependencies
    npm install
  2. Compile the Win32 helper
    g++ -municode HenryAppBar.cpp -o HenryAppBar.exe -lole32 -lshell32 -lgdi32
  3. Build installer / portable EXE
    nvm use 18.20.5
    npm run dist

Development

Run Henry in development mode:

npm start
